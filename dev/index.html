UnitfulEquivalences.jl

This package extends Unitful.jl to allow conversions between equivalent quantities of different dimension, e.g., between energy and wavelength of a photon.

Installation

This package is compatible with Julia ≥ 1.0 and Unitful ≥ 1.0. It can be installed by typing

] add UnitfulEquivalences

in the Julia REPL.
